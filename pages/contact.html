Common Questions

Quick answers about this website.

❓

Is this the official SLC website?

No, this is a fan website created as an ICT 142-3 academic assignment at Uva Wellassa University. For official information, visit srilankacricket.lk.

📊

Are the statistics accurate?

All statistics have been carefully researched from public cricket records and ESPNcricinfo. Some current player stats may vary slightly over time.

🖥️

How was this website built?

Built using only HTML, CSS, and vanilla JavaScript — no frameworks or libraries. Designed to be responsive for mobile and desktop.
